DESIGN PRINCIPLES

Built on three hard commitments.

⚖

Governance-first

Every action is logged, scoped, and reviewed. It asks before touching important code and catches its own mistakes before you do. You don't have to watch it — it watches itself.

◈

Self-solving architecture

When something fails, it tries different approaches. When it learns something new, it remembers for next time. Improves with every session.

◉

Human-in-control

You set the autonomy level — fully autonomous, approval-required, or advisory-only. Pause, redirect, or override at any point.

HOW IT WORKS

Three stages. One command.

Every task goes through the same disciplined process — no shortcuts, no guessing.

◈

Understand

Analyzes your codebase, maps dependencies, identifies what matters before touching anything. Plans tasks, defines acceptance criteria, and confirms scope with you on consequential work.

⊞

Execute

Decomposes work into parallel tasks. Independent workstreams run simultaneously with conflict-free coordination. Every task starts with a failing test and ends with passing verification.

◉

Verify

Multi-stage adversarial review. Every change is tested, traced, and independently verified before you see it. Nothing ships with stubs, broken pipes, or unverified claims.

SYSTEM CAPABILITIES

Six things built in from day one.

◉

Persistent Memory

Learns from every session. Recalls similar bugs, successful patterns, and past decisions. The more you use it, the faster it gets.
⊞

Parallel Execution

Independent tasks run simultaneously. Conflict-free coordination keeps quality high even at speed.
⌘

Self-Governing

Built-in quality enforcement. Asks before touching critical code. Reviews its own work before you see it.
≡

Test-Driven

Every task starts with a failing test. No code ships without passing verification. No stubs. No shortcuts.
⟳

Model Freedom

Works with any LLM — Claude, GPT, Gemini, Llama, or your own. Switch providers without changing workflows.
◈

Zero Infrastructure

Runs in your terminal. No daemon. No Docker. No cloud account. One command to install.
